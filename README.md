# ADITON Mimarlık Ofisi Website

A custom one-page architecture portfolio for ADITON Mimarlık Ofisi, built with HTML5, CSS3 and vanilla JavaScript only.

## Structure

- `index.html`
- `css/style.css`
- `js/script.js`
- `assets/logo.png`
- `assets/hero.jpg`
- `assets/project1.jpg`
- `assets/project2.jpg`
- `assets/project3.jpg`

## Features

- Responsive desktop, tablet and mobile layout
- Sticky transparent-to-solid navigation
- Intersection Observer reveal animations
- Scroll spy navigation
- Animated counters
- Project modal gallery
- Minimal testimonial slider
- Contact form validation
- Back-to-top control
- SEO metadata and structured data
- Lazy-loaded project imagery

## Deploy

Upload this folder to GitHub Pages or any static host. No build step is required.
